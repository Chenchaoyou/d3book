5 D3 scripts

To follow along with the code in this section, download and open a fresh copy of six_blue_circles.html, or use the online version.

5.1 Adding D3 to html file

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>Change the title!</title>
    <script src="https://d3js.org/d3.v5.min.js"></script>  <!-- link to D3 library -->
  </head>

  <body>
    <script>
    // JavaScript / D3 will go here
    </script>
  </body>

</html>

5.2 Add svg

<script>
d3.select("body").append("svg").attr("width", "500").attr("height", "400");
</script>

5.3 Add a circle

<script>
d3.select("body").append("svg").attr("width", "500").attr("height", "400");

d3.select("svg").append("circle").attr("cx", "200").attr("cy", "250")
  .attr("r", "20").attr("fill", "red");
</script>
